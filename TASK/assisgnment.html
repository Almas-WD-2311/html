• Website is a place connected to internet. where company, organisation, etc. puts information that can be found on the world wide web. central location with more than one web pages or several web pages.
  • WWW it is world wide web. Collection of websites or web pages stored in web servers and connected to local computers through the internet.
  • HTTP/HTTPs HTTP stands for Hypertext Transfer protocol. when the user makes an http request on the browser, then the webserver sends the requested data to the user in the form of web pages. HTTPs stands for Hypertext Transfer protocol Secure. HTTPs protocolis mainly required where we need to enter the bank details. The HTTPs protocol is secured due to the SSL protocol.
  • Client server Cient-server is a relationship in which one program, the client, requests a service or resource from another program, the server.
  • Domain A domain name is the name of your website. the system by which your domain name is operated or run is called domain name system,it is also called DNS in short.
  • Host Any computer connected to a network. It can provide information, applications or services to other hosts or nodes on the network.
  • Hosting Web hosting is an online service that makes your website's content accessible on the internet. Web host is a space where we store all data which we want to share to the world any time and that is called through an IP address as well as Domain name.
  • FTP File Transfer Protocol. standard network protocol used for the transfer of files from one host to another over a TCP-based network, such as the Internet.
  • Path A path is either relative or absolute. An absolute path always contains the root element and the complete directory list required to locate the file. For example, /home/sally/statusReport is an absolute path. All of the information needed to locate the file is contained in the path string.
  • Extension an extension is a file containing programming that serves to extend the capabilities of or data available to a more basic program. Extensions are sometimes required to be stored in a separate extensions file so that they're easy to locate.
  • Editor / vscode software programs that enable the user to create and edit text files. Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS.
  • URL-URI/URN A URL (Uniform Resource Locator) is a unique identifier used to locate a resource on the Internet. Uniform Resource Identifier (URI) is a unique sequence of characters that identifies a logical or physical resource used by web technologie. (Uniform Resource Name) is a permanent identifier for internet resources and employs the urn scheme.
  • Address Bar An address bar may be referred to as a location bar, address box, or URL bar. An Internet browser's address bar is a feature to enter and display a website's address.
  • Title Bar Title Bar in the computer is defined as a horizontal bar. This horizontal bar is located at the topmost part of a window within GUI. It shows the software title that is currently running.
